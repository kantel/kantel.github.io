Spät kommen sie, aber sie kommen – die Videotutorials zum Wochenende, die ich jede Woche für Euch raussuche, damit über die Feiertage nicht aus lauter Langeweile in die Kirchen rennt. Natürlich ist heute immer noch Trinket das große Thema. Denn irgendwie muß ich meine (Wieder-) Entdeckung dieser Online-IDE (nicht nur) für Python doch feiern. Aber auch ein paar andere Topics stehen auf der heutigen Agenda.

Doch erst einmal Trinket: Der britische Kanal mit dem unaussprechlichen Namen srogersps hat einen Python Turtle Trinket Course im Angebote. Er besteht aus 23 Videos, die meisten sind jedoch – da der Kanal aus Unterrichtsmaterial besteht – kurze Übungsaufgaben. Wichtig (und jeweilige Zusammenfassung) sind daher nur diese vier je etwa fünfzehnminütigen Videos:

  • KS3 Python Turtle 1.0 - Presentation Complete
  • KS3 Python Turtle 2 Presentation Complete
  • KS3 Python Lesson 3 Presentation Complete
  • KS3 Python Turtle Lesson 4 Presentation Complete

Und von dem Kanal STEM Crafter gibt es unter anderem das Video Using Images in Python Trinkets. Aber dort gibt es noch mehrere Tutorials zu Trinket, so zum Beispiel Turtle Graphics Update (15 Minuten) oder Python - Gaming - Solution Video (12 Minuten), aber leider keine Playlist. Ich habe den Kanal daher abonniert.

Doch jetzt zu anderen Themen: Auf Franks Laboratory gibt es einen dreiviertelstündigen »Creative coding CRASH COURSE for beginners« ganz ohne externe Frameworks nur mit HTML5 Canvas und Vanilla JavaScript. Das Laboratorium ist sowieso eine Fundgrube hochwertiger, meist sehr umfangreicher Tutorials zu JavaScript, HTML und CSS (alles ohne externe Bibliotheken). Empfehlen kann ich hieraus eigentlich alles, aber die Playlists

  • Generative Art for Beginners mit vier Videos und insgesamt dreistündiger Spieldauer (daraus ist das obige Video),
  • JavaScript Game Development Masterclass 2022 mit 19 Videos je zwischen 15 Minuten und mehr als drei Stunden Spieldauer (daraus ist auch das dreistündige Video »JavaScript Isometric Physics Game Tutorial – siehe Bannerbild oben),
  • Front End Web Development Projects for Beginners, eine Monsterplaylist mit 37 Videos, fast alle mehr als eine Stunde lang, und
  • Game Development with Vanilla JavaScript mit sechzehn ebenfalls sehr langen Videos

verdienen eine besondere Beachtung. Natürlich gibt es bei den einzelnen Playlists Doppelungen und Überschneidungen, aber die Entdeckerfreude ist das wert.

Noch ein Laboratorium: Ich plane mit Quarto noch etwas mehr anzustellen, als nur dieses Blog Kritzelheft herauszuschreiben. Daher kam mir der einstündigen Lehrgang zu »Quarto Markdown Basics« aus Crump’s Computanional Cognition Lab wie gerufen.

Und zum Abschluß noch ein wenig Computergeschichte(n): Another Boring Topic präsentiert »The Rise of Windows Part 1 (1981-1985)«. Teil 2 wird (eventuell – sagen die Macher) folgen.

Nun ein schönes Wochenende Euch allen da draußen. Ich will mich über die nächsten Tage tiefer in Trinket einwühlen und habe also zu tun. Wir lesen uns aller Voraussicht daher erst am Montag wieder. Still digging!
