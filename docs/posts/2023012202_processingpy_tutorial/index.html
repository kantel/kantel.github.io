Make, Code, Have Fun with Processing.py

Python
Processing.py
Autor:in

Jörg Kantel

Veröffentlichungsdatum

22. Januar 2023

Nur ganz kurz, bevor ich mich wieder in der Erstellung interaktiver Geschichten verliere: Es gibt einen YouTube-Kanal mit dem schönen Namen »Make, Code, Have Fun«, der genau das vermitteln will, was der Namen verspricht. Und auf diesem Kanal ist nun ein schönes Tutorial zu Processing.py erschienen:

Da der Kanal für Kinder gedacht ist, meint YouTube – warum auch immer – das es eine gute Idee sei, daß Speichern der Videos in eigene Playlists (zum Beispiel »Später ansehen«) zu verhindern. Daher hier die Playlists, die ich interessant finde und die ich mir »später« einmal ansehen möchte:

  • Make Code Python 101 besteht aus sechs Tutorials mit einer Laufzeit zwischen je fünf Minuten und einer Viertelstunde (die letzte Aktualisierung erfolgte am 10. Januar 2021).
  • Python Basic ist eine bedeutend umfangreichere Playlist mit sechzehn Videos zwischen je fünf Minuten und einer halben Stunde, die zuletzt am 14. Januar 2022 aktualisiert wurde.
  • Und die Playlist Python 2 ist eine Art Gemischtwarenladen, der von der Turtle über Pygame Zero und die Matplotlib bis eben hin zu Processing.py so ziemlich alles behandelt, was auch nur entfernt nach der Schlange riecht. Gerade deswegen ist sie vermutlich besonders interessant. Sie besteht aus 18 Videos unterschiedlicher Länge (von ganz kurz bis fast eine Stunde) und wurde zuletzt heute aktualisiert.

Gemäß ihrem Anspruch, Kindern das Programmieren beizubringen, setzen die Macherinnen und Macher auf den Editor Mu, aber auch Thonny ist sicher geeignet, um die meisten Projekte zu implementieren.
